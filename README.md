## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Customization

- Update `src/components/` files to change content.
- Modify `src/index.css` to change the color scheme.
